# Privacy Policy

**Easy Markdown** is a local-first markdown editor. Your documents stay on your device.

## Data we collect

None. Easy Markdown does not collect, transmit, or store any personal data, usage analytics, crash reports, or telemetry. There is no account system. There is no sign-in.

## Data we access

Easy Markdown only accesses files you explicitly open through your operating system's file picker. The app does not browse your file system and does not access files you have not selected. Files you open from a cloud drive (iCloud Drive, Dropbox, Google Drive, OneDrive) are accessed through your OS's file providers — Easy Markdown has no separate cloud integration.

## Data we store

- **Your open file**: held in memory while editing, saved back to its original location on disk when you choose to save.
- **App settings**: font, theme, editor preferences. Stored in a local config file next to the app's data directory for the platform. Not synced. Not transmitted.
- **Recent files list**: a small list of paths to recently opened files, stored locally.

Easy Markdown never writes sidecar files or proprietary metadata into your markdown files. `.md` in, `.md` out.

## Network access

The core editor makes zero network requests. No content leaves your device.

**Optional BYO-key AI**: if you paste an API key for a third-party AI provider (OpenAI, Anthropic, Ollama, or any OpenAI-compatible endpoint) in settings, Easy Markdown will make AI requests *directly from your machine to that provider*. Easy Markdown never proxies these requests, never stores your key anywhere except encrypted local storage on your device, and never sees the content you send. The privacy of that content is governed by the third-party provider you chose.

## Third-party services

Easy Markdown uses no third-party SDKs, analytics services, ad networks, crash reporters, or telemetry libraries. The only time any network traffic leaves your device is if you have opted into BYO-key AI and you explicitly invoke an AI feature — and in that case the traffic goes directly to the provider you chose, not through us.

## Children's privacy

Easy Markdown does not collect information from anyone, including children under 13.

## Changes

Privacy policy changes will be published in this file on the project's public repository. There is no app-based changelog because there is no app-to-server connection.

## Contact

For privacy questions, open an issue on the project's GitHub repository.
