# Privacy Policy

**Easy Markdown** is a local-first markdown editor. Your documents stay on your device.

**Last updated:** 2026-03-29

## Data We Collect

None. Easy Markdown does not collect, transmit, or store any personal data, usage analytics, crash reports, or telemetry.

## Data We Access

Easy Markdown accesses only the files you explicitly open via the system file picker. File access is scoped to individual documents using Apple's security-scoped bookmarks. The app cannot browse your file system or access files you have not selected.

## Data We Store

- **Your open file**: held in memory while editing, saved back to the original location on disk.
- **App settings**: font choice, theme, editor preferences. Stored in UserDefaults on your device. Not synced or transmitted.
- **Recent files list**: filenames and bookmark data for up to 20 recently opened files. Stored in UserDefaults on your device.

## Network Access

The core editor makes zero network requests. No data leaves your device.

## Third-Party Services

Easy Markdown uses no third-party SDKs, analytics services, ad networks, or crash reporters.

## Children's Privacy

Easy Markdown does not collect information from anyone, including children under 13.

## Changes

If this policy changes, the updated version will be published in the app and at the URL provided in App Store Connect.
